# Serangic's Dotfiles

### Pre-requisite work

<p>The various configuration files that are distributed in this repository only really work in tandem with my machine, and my organisation. An example would be i3blocks, which relies on a directory called scripts, which houses a bunch of other stuff... It's basically i3blocks-contrib cloned and with some additional stuff. TLDR, if you want to use my configuration, get ready to re-organise stuff that I am too lazy to organise for you, because these are meant to be personally for me.</p>
